プライバシーポリシー

最終更新日: 2026-04-11

「Junkpot」（以下「本アプリ」といいます）の運営者（以下「当方」といいます）は、本アプリにおけるユーザーの個人情報および利用情報の取り扱いについて、本プライバシーポリシー（以下「本ポリシー」といいます）を定めます。

1. 取得する情報

本アプリでは、サービス提供のために以下の情報を取得・利用する場合があります。

カテゴリ内容取得の根拠
端末情報OS バージョン、機種識別子（IDFV / Android ID 等の匿名識別子）クラッシュ・不具合解析、不正利用防止
購入情報購入トランザクション ID、購入アイテム種別、購入日時アプリ内課金の付与・復元
ゲーム進行情報スコア、所持メダル数、進行状況端末内ローカル保存、グローバルランキング表示（実装時）
広告関連情報広告 ID（Apple: IDFA / Google: GAID）広告配信およびその効果測定（ユーザーが追跡許諾した場合のみ）

本アプリは氏名、住所、電話番号、メールアドレスといった一般的な「個人情報」を、お問い合わせ等で直接ご提供いただく場合を除き、自動的に取得することはありません。

本アプリは以下の第三者サービスを利用します。各サービスのプライバシー方針もあわせてご確認ください。

3. 取得した情報の利用目的

  1. 本アプリの提供および機能改善
  2. アプリ内課金の付与・復元およびその技術的サポート
  3. クラッシュ・不具合の原因調査
  4. 不正利用・チート行為の検知および防止
  5. ユーザーサポート対応
  6. 法令または公的機関からの正当な要請への対応

4. 第三者への提供・開示の制限

当方は、次のいずれかの場合を除き、ユーザーの情報を第三者に提供しません。

  1. ユーザーの同意がある場合
  2. 法令に基づく場合
  3. 人の生命・身体・財産の保護のために必要があり、ユーザーの同意を得ることが困難であるとき
  4. 上記「第三者提供・委託先」に記載した、本アプリの運営に必要な範囲での委託

5. 保存期間

購入トランザクション IDユーザー端末内重複付与防止のため恒久
広告 ID（許諾時）広告事業者の管理下広告事業者の方針による

6. ユーザーの権利

ユーザーは以下を行うことができます。

  • 端末の OS 設定で広告 ID をリセットまたは追跡を拒否すること
  • 端末の OS 設定で本アプリの通知を無効化すること
  • 本アプリを端末からアンインストールすることで、ローカルに保存されたすべてのゲーム進行情報を消去すること

ランキング基盤の導入後は、ランキングに紐づく自身の匿名 ID の削除リクエストを当方の連絡先まで送付できます。

7. 13 歳未満の利用について

本アプリは、Apple App Store の 17+ 相当のレーティングで配信されており、13 歳未満の利用を主たる対象としていません。13 歳未満のお子様が本アプリを利用される場合は、保護者の方の同意のもとでご利用ください。

8. 改定

当方は、必要に応じて本ポリシーを改定することがあります。重要な変更がある場合は、本アプリ内またはストアの説明文にて告知します。
